# Portfolio Website — Muhammad Rizki Hidayat

**Live:** https://portfolio-muhammadrizkihidayat.vercel.app/

Website portofolio pribadi satu halaman yang menampilkan profil profesional, keahlian, proyek, dan kontak. Dirancang sebagai media personal branding untuk merepresentasikan kompetensi pada bidang **Data Analysis** dan **Web Development** dengan tampilan modern, responsif, serta dukungan dark/light theme.

## Fitur Utama

* Profil dan deskripsi profesional
* Daftar skill teknis dan tools
* Showcase proyek pengembangan sistem dan analisis data
* Form kontak dengan pengiriman email otomatis
* Konten dinamis melalui satu file data

## Teknologi

HTML5 • CSS3 • JavaScript • EmailJS • Vercel

## Struktur Proyek

portfolio/
├── index.html
├── style.css
├── script.js
├── data.js
└── assets/

## Menjalankan Secara Lokal

1. Download atau clone project.
2. Buka folder project.
3. Jalankan:

   * Klik langsung **index.html**, atau
   * Gunakan VS Code → extension **Live Server** → Open with Live Server.

## Mengubah Konten

Semua isi website diatur dari satu file: **data.js**.
Perubahan nama, deskripsi, proyek, gambar, dan kontak cukup diedit di file tersebut lalu refresh halaman.

## Form Kontak

Form mengirim pesan langsung ke email pemilik melalui EmailJS dengan data: nama pengirim, email, subjek, dan isi pesan.

## Tujuan

Sebagai media personal branding, dokumentasi karya, serta sarana komunikasi profesional berbasis web.
